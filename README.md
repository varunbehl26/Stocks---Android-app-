# Stocks---Android-app-
 Yahoo stocks API

Fully functional android app built from start for Android Developer Nanodegree program. Features With the app, you can: 

• View stocks details

• Add new stocks

• View history of stocks via graph

Libraries used 

• Retrofit

• Dagger
